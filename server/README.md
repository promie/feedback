# Feedback Server API

## Requirements
- Node 14.17.4 LTS
- NPM 6.x

## Installation
```bash
npm install
```

## How to run
### Local
To run the both apps in development mode, run the below command on the root directory:
```bash
npm run dev
```

An express server will spin up on `http://localhost:5000/`

## How to run tests
Run all tests for server app
```bash
npm run test
```
